# 01 — Installing VirtualBox

VirtualBox is the hypervisor used for running the virtual machines.

## Step 1: Download VirtualBox
1. Visit the official website:  
   https://www.virtualbox.org/wiki/Downloads
2. Select the installer for your OS (Windows / macOS / Linux).
3. Download the latest stable version.

## Step 2: Install VirtualBox
1. Run the installer.
2. Accept default settings.
3. Allow network interface installation when prompted.
4. Finish installation and launch VirtualBox.

## Step 3: Install VirtualBox Extension Pack
1. On the same downloads page, scroll to “Extension Pack”.
2. Download the version that matches your VirtualBox version.
3. Double-click `.vbox-extpack` to install.
4. Accept the license agreement.

VirtualBox is now ready for creating the lab.
